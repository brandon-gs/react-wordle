# Wordle

Wordle gama made with Typescript, React and Tailwind.

## Getting Started

1.- Clone the repository

```bash
$ git clone git@github.com:brandon-gs/react-wordle.git
```

2.- Install the dependencies

```bash

$ npm i -g pnpm
$ pnpm install

# $ npm install

# $ npm i -g yarn
# $ yarn

```

3.- Run the project locally

```bash
$ pnpm dev

# $ npm run dev

# $ yarn dev
```
